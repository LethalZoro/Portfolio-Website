## 8. Projects

> Projects with full descriptions come from the CV and the current site. Live/repo links included where known. Existing site images are noted so they can be reused.

### Featured — AI / ML

#### Contract Summary AI
Full-stack app (Flask + React) that summarizes and analyzes UK construction contracts.
- Backend RESTful API in Flask integrated with LangChain, OpenAI GPT-4o, and ChromaDB for document processing and summarization.
- Retrieval-Augmented Generation (RAG) to generate detailed summaries and answer user queries over uploaded PDF contracts.
- Responsive React frontend for uploading documents, viewing summaries, and chatting with the AI assistant.
- **Live:** https://react-contract-chatbot.vercel.app/ · **Repo:** https://github.com/LethalZoro/Contracts-summary-Flask · **Image:** `/contract-summary.png`
- **Tech:** Flask, React, LangChain, OpenAI GPT-4o, ChromaDB, RAG, Vercel

#### LangGraph Chatbot Workflow
Interactive chatbot system built on LangGraph, LangChain, and a local LLM.
- Dynamically manages conversation flows based on user input and state management.
- Real-time client–server communication over WebSockets with instant state synchronization.
- Responsive frontend (HTML/CSS/JS) visualizing conversation history and structured JSON state.
- **Live:** https://langgraph-chatbot-wtth.onrender.com/ · **Repo:** https://github.com/LethalZoro/LangGraph-Chatbot · **Image:** `/langgraph-ai.png`
- **Tech:** LangGraph, LangChain, local LLM, WebSockets, Render

#### Federated Learning Movie Recommendation System
Privacy-preserving movie recommender using Federated Learning + Collaborative Filtering.
- User-Based (UBCF) and Item-Based (IBCF) Collaborative Filtering on the MovieLens 20M dataset.
- Federated Learning keeps user data on-device — only model updates are shared, never raw data.
- Addressed scalability and cold-start challenges common to large-scale recommenders.
- **Repo:** https://github.com/LethalZoro/Federated-Recommender-System · **Image:** `/Federated-Recommender-System.png`
- **Tech:** Federated Learning, Collaborative Filtering, Python

#### Reinforcement Learning Mario Agent
Trained a Gym Mario agent with several RL algorithms.
- Implemented DQNN, CNN, and MobileNet-V2 approaches; best results achieved with a Deep Q-Neural Network (DQNN).
- **Repo:** https://github.com/LethalZoro/Gym-Mario-RL · **Image:** `/mario.png`
- **Tech:** Reinforcement Learning, DQN, PyTorch, OpenAI Gym

#### Vision Transformer (CIFAR-10)
Image-classification model built with a Vision Transformer.
- Built a ViT to classify the 10 CIFAR-10 classes, leveraging self-attention for vision.
- Implemented with PyTorch and the Hugging Face API.
- **Tech:** PyTorch, Hugging Face, Vision Transformer *(link to confirm)*
